# **Единый стиль мышления: "Проблема -> Решение -> Реализация"**"


Этот файл - основа любого взаимодействия. 

Прежде чем писать код, агент должен:* **Четко сформулировать проблему или задачу** (что нужно сделать, почему).

* **Составить краткий план решения** (какие компоненты/функции потребуются, какие технологии/библиотеки использовать, какие могут быть подводные камни).
* **Только потом приступать к реализации.**
